Composable функция для отображения списка упражнений.

Эта функция использует ViewModel для управления состоянием списка упражнений и отображает список упражнений, а также всплывающее окно для добавления или редактирования упражнения.

Эта функция отображает список упражнений, используя LazyColumn для эффективного отображения больших списков. Она также обрабатывает события, связанные с добавлением или редактированием упражнения, через всплывающее окно.

Эта функция отображает информацию об одном упражнении, включая его название, описание, вес и флаг "Следующее". Она также позволяет пользователю нажать на элемент списка для редактирования упражнения.

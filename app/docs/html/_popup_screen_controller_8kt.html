Detailed Description

Composable функция для отображения всплывающего окна добавления упражнения.

Эта функция использует ModalBottomSheet для отображения всплывающего окна, позволяющего пользователю добавить новое упражнение. Она включает в себя поля для ввода названия, описания, типа упражнения и веса (или расстояния для кардио упражнений).
